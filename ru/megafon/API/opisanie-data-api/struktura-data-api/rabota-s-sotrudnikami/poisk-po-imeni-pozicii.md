### `GET /position?name={name}`

**Описание:** метод позволяет получить параметры должности по её названию.

## Параметры метода

{% table %}

---

*  {% isHeader=true %}

   Название

*  {% isHeader=true %}

   Тип

*  {% isHeader=true %}

   Допустимые значения

*  {% isHeader=true %}

   Описание

---

*  {% isHeader=true %}

   name

*  string

*  Произвольный текст

*  Название должности

---

*  {% isHeader=true %}

   id

*  integer

*  Целое число

*  Уникальный идентификатор должности.  Получается в качестве ответа при выполнении запроса.

{% /table %}

## **Структура запроса**

`GET /position?name=Нода 0`

## **Структура ответа**

```json
[
    {
        "name": "Нода 0",
        "id": 81
    }
]
```
