### `GET /position`

**Описание:** этот метод используется для получения списка всех должностей в системе.

*  string

*  Произвольный текст

*  Название должности

---

   id

*  integer

*  Целое число

*  Уникальный идентификатор должности

## Cтруктура запроса

`GET /position`

```json
[
    {
        "name": "Test",
        "id": 45
    },
    {
        "name": "Нода 0",
```
