### `DELETE /scenario/outbound/regional/{id}`

**Описание**: метод позволяет удалить региональный исходящий сценарий.

## Структура запроса

`DELETE /scenario/outbound/regional/533`

## Cтруктура ответа

```json
{
   "id": 533
}
```
