### `DELETE /numbers/virtual-number/{id}`

**Описание**: удаление виртуального номера

:::note Внимание!

Удаление городских номеров невозможно

:::

## Структура запроса

`DELETE /numbers/virtual-number/101`

## **Структура ответа**

```json
{
   "id": 101
}
```
