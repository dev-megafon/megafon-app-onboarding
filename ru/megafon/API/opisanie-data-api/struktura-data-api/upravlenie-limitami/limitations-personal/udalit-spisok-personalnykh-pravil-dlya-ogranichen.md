### `DELETE /limitations/personal/{id}`

**Описание**: метод позволяет удалить персональные правила сотрудников или отделов для ограничения звонков.

## Структура запроса

`DELETE /limitations/personal/63`

## Структура ответа

```json
{
   "id": 63
}
```
